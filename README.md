# AI Readiness Diagnostics

Most teams don't have an AI problem first. They have a workflow, environment, and decision-quality problem.

This toolkit helps leaders, team leads, and internal AI champions answer four practical questions before spending money or time on AI:

1. **Should we build, buy, or fix the basics first?**
2. **Is our workflow actually ready for AI?**
3. **Is this vendor proposal worth paying for?**
4. **Where should we focus so effort compounds?**

## How to use the prompts

1. Copy the full prompt text
2. Paste it into any AI assistant
3. Answer the questions one at a time — the AI will guide you through
4. Get a scored, actionable output

The prompts are designed to push for specifics, score only what you actually tell them, and give routing decisions — not vague observations.

## Framework

All four diagnostics share the same [operating logic](tim-diagnostic-framework.md):

- Ask one question at a time
- Push for specifics — if the answer is vague, say so
- Separate technical, workflow, leadership, and domain/regulatory layers
- Score evidence strength (High / Medium / Low)
- End with a decision, a first-action list, and what NOT to do
